Migasfree Play
==============

Migasfree Client front-end. Allow install/uninstall available applications and printers.

Requirements
============

    * Node Webkit
    * migasfree-client


Installation
============

$ cd deb
$ ./make-deb.sh
$ sudo ../dist/dpkg -i migasfree-play_X.X-X_all.deb
